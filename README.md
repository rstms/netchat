# netchat

netcat(socat) + chat

------

## Requirements

### python modules
  - `click`
  - `pexpect`

### command line tools
  - `socat`

------

## Usage:
```
Usage: netchat [OPTIONS] ADDRESS PORT [SCRIPT]

Options:
  -f, --file FILENAME
  -t, --timeout INTEGER
  -v, --verbose
  -e, --echo
  -d, --debug
  --help                 Show this message and exit.
```
